# Active Directory / Group Policy Lab

**Project Overview**: Built a Windows Server 2025 Active Directory Domain Controller lab in VMware. Configured OUs, users, groups, and GPOs.

## Objective
Set up a full Active Directory domain from scratch on Windows Server 2025.

## Setup Steps
1. Download Windows Server 2025 evaluation ISO
2. Install in VMware Workstation Pro / VirtualBox
3. Promote server to Domain Controller (`example.local`)
4. Install AD DS, DNS, and Group Policy Management
